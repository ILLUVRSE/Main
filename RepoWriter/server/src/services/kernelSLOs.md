# Operational SLOs for Kernel
## Service Level Objectives (SLOs)
- **Availability**: 99.9%
- **Latency**: 95th percentile response time under 200ms
- **Error Rate**: Less than 1%
## Monitoring
- **Metrics**: Track availability, latency, and error rates.
- **Alerts**: Set up alerts for SLO breaches.
## Incident Runbooks
1. **Incident Detection**: Monitor alerts for SLO breaches.
2. **Incident Response**: Follow the incident response plan.
3. **Postmortem**: Conduct a postmortem analysis after resolution.
